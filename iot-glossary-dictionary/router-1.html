Router

May 10, 2022 Roel van der Meer

A router is a networking device that forwards data packets between computer networks. Routers perform the traffic directing functions on the Internet. Routers determine whether the source and destination are on the same network or if the data being transmitted must be transferred from one network type to another. This process requires encapsulating the data packet with routing protocol header information for the new network type and then forwards the data packets accordingly.

Routers are typically box-shaped devices that connect directly to computers via "hard-lined" or wired connections. One connection port on the router allows the router to connect to a modem for receiving Internet data packs, while another set of ports allows a router to connect to computers for distributing Internet data packets. Some routers also provide ports for distributing data packets to fax machines and telephones. The Router is a layer 3 (Network layer) device in the Open Systems Interconnection (OSI) model, and are responsible for assigning a local IP address to each of the devices on the network. The essential difference between a Switch and a Router is that switch is designed to connect computers within a network, while a router is designed to connect multiple networks together.
