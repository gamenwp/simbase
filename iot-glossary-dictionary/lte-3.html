Long-Term Evolution (LTE)

May 10, 2022 Roel van der Meer

Long-Term Evolution (LTE) is a standard in wireless communication developed by the 3rd Generation Partnership Project (3GPP) for mobile devices and data terminals. LTE uses new Digital signaling processes (DSP) to make it ten times faster than 3G networks. LTE is regarded as an evolution in 3G and is sometimes referred to as 3.95G (though it is marketed as 4G LTE, it does not satisfy the requirements of a true 4G wireless service). LTE uses Orthogonal Frequency Division Multiplex (OFDMA) radio interface instead of the CDMA used in 3G, therefore connected devices need to be able to accept transmission from those radios. Also, in LTE network, the core GPRS network is replaced with Evolved Packet Core (EPC) which supports four times the data and voice capacity and provides a more seamless handover process between cells.

LTE uses 2 radios. One for uplink and another for downlink. This provides a two-way communication for network optimization. LTE uses Orthogonal Frequency Division Multiple Access (OFDMA) for downlink to cope with increased resource demand from the cell to the subscriber. OFDMA mandates Multiple Input Multiple Output (MIMO) which means devices have multiple connections to a given cell to provide network stability, reduce latency and increase the total throughput of the network. For the uplink, LTE uses a Single Carrier Frequency Division Multiple Access (SC-FDMA). This radio also supports MIMO0 and is better for uplink as it has a better peak-to-average power ratio. LTE uses weaker signals to transmit back to the cell to conserve battery life of end devices.
