Cellular network

May 11, 2022 Roel van der Meer

Cellular Network also referred to as a mobile network is a network of radios distributed over land. These networks are distributed by cells. These cells contain at least one fixed transceiver which provides network coverage for the range in which it covers. In cellular communication, neighboring cells use different sets of frequencies to avoid interference. A collection of cells provides a wholistic network coverage for a mobile network. This provides a means in which end user mobile devices can communicate even when moving through cells.

One key characteristic of a cellular network is the ability to re-use frequencies, providing an increased coverage and capacity. The factors that govern frequency reuse are the reuse distance, which states the distance between cells – and the reuse factor – which is the rate at which a frequency can be used in a network. To distinguish cellular signals from other transmitters, cellular networks use a variety of technology which include Frequency Division Multiple Access (FDMA), Time Division Multiple Access (TDMA) and Code division Multiple Access (CDMA).

Cellular networks function in a hierarchical system. A cellular network consists of a Base station transceiver (BTS) which routes calls from mobile devices to its corresponding Base station controller (BSC). The BSC works together with the Mobile Switching center (MSC) to interface with the public switched telephone network (PSTN), the visitor location register (VLR) and the home location register (HLR) to route calls over different base stations.
